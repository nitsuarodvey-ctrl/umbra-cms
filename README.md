# Umbra Engine | AI-Driven Web Architect 🌌

[TR] Umbra, yerel bir yapay zeka (LM Studio) ile konuşarak saniyeler içinde tam fonksiyonel, içerikleri AI tarafından üretilmiş web siteleri oluşturan bir "AI-to-UI" motorudur.
[EN] Umbra is an AI-to-UI engine that generates fully functional, AI-content-driven websites in seconds by communicating with a local AI (LM Studio).

---

## ⚡ Özellikler / Features

### 🇹🇷 Türkçe
- **Local AI Entegrasyonu:** LM Studio üzerinden yerel ağda güvenli ve sınırsız içerik üretimi.
- **Anlık Enjeksiyon:** AI'dan gelen JSON verisini `postMessage` ile iframe şablonlarına anlık basma.
- **3 Farklı Ruh:** Creative (Ajans), Startup ve Corporate (Kurumsal) şablon desteği.
- **Temiz Çıktı:** Üretilen sitenin kodlarını CSS ve HTML olarak tek tıkla indirme.

### 🇺🇸 English
- **Local AI Integration:** Secure and unlimited content generation via LM Studio on local network.
- **Instant Injection:** Real-time injection of AI-generated JSON into templates via `postMessage`.
- **Triple Templates:** Support for Creative (Agency), Startup, and Corporate design modes.
- **Code Export:** One-click download of generated sites as clean HTML/CSS files.

---

## 🛠️ Teknik Altyapı / Tech Stack

- **Logic:** Vanilla JS (API Fetch & JSON Manipulation)
- **Styling:** Tailwind CSS & Glassmorphism
- **Library:** Font Awesome & Google Fonts
- **Engine:** LM Studio (Local LLM Server)

---

## 🚀 Nasıl Çalışır? / How to Run?

1. LM Studio'da bir model (Llama, Mistral vb.) yükleyin ve Server'ı (Port 1234) başlatın.
2. `admin.html` dosyasını tarayıcıda açın.
3. Marka ismini ve detaylarını girip "OLUŞTUR" butonuna basın.
